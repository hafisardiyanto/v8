# Laravel Project v8

## Overview
This is a Laravel project designed for employee attendance management. It provides a user-friendly interface for logging in and managing attendance records.

## Directory Structure
- **app**: Contains the core application code, including models, controllers, and business logic.
- **bootstrap**: Contains files for bootstrapping the application, including the application instance and configuration loading.
- **config**: Contains configuration files for various aspects of the application, such as database connections and mail settings.
- **database**: Contains database migrations, seeders, and factories for the application.
- **public**: Contains the front-facing files of the application, including the main entry point (index.php), assets like CSS and JavaScript files, and images.
- **resources**: Contains resources such as views, raw assets (Sass, JavaScript), and language files.
  - **js**: JavaScript files for the application.
  - **lang**: Language files for localization.
  - **sass**: Sass files for styling.
  - **views**: Blade templates for rendering views.
    - **auth**: Authentication-related views.
      - **login.blade.php**: Blade template for the login page.
- **routes**: Contains route definitions for the application, mapping URLs to controllers.
- **storage**: Contains files generated by the application, such as logs, cached views, and file uploads.
- **tests**: Contains automated tests for the application, including unit and feature tests.
- **artisan**: Command-line interface for the Laravel application.
- **composer.json**: Configuration file for Composer, listing dependencies and scripts.
- **package.json**: Configuration file for npm, listing JavaScript dependencies and scripts.
- **phpunit.xml**: Configuration file for PHPUnit, specifying settings for running tests.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd v8
   ```
3. Install dependencies:
   ```
   composer install
   npm install
   ```
4. Set up your environment file:
   ```
   cp .env.example .env
   ```
5. Generate the application key:
   ```
   php artisan key:generate
   ```
6. Run migrations:
   ```
   php artisan migrate
   ```

## Usage
To start the local development server, run:
```
php artisan serve
```
Visit `http://localhost:8000` in your browser to access the application.
